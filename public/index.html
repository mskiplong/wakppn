Manajemen Kontak

Kelola kontak dengan x1 (nomor) dan x2 (grup)

Format Excel

File Excel harus memiliki kolom minimum: x1 (nomor WhatsApp) dan x2 (nama grup).

Catatan: Data kontak juga bisa ditambahkan otomatis dari upload di Blast Massal dan Reminder.

x1 (Nomor WA) x2 (Grup) x3 x4 Aksi
Belum ada kontak. Upload file Excel atau upload dari Blast/Reminder.

Kirim Pesan Single

Kirim pesan ke satu nomor WhatsApp

Variabel yang tersedia:

{x1} - Nomor WA
{x2} - Nama Grup
{x3} - Kolom 3
{x4} - Kolom 4
{x5} - Kolom 5

Blast Pesan Massal

Kirim pesan ke banyak kontak sekaligus

Langkah Blast Massal

1. Upload file Excel (x1 & x2) → Data otomatis masuk ke Manajemen Kontak
2. Pilih grup → 3. Pilih konsep → 4. Kirim

Step 1: Upload Data Excel

Klik atau Drag file Excel (.xlsx)

Format: Kolom x1 (nomor) dan x2 (grup)

Data akan otomatis masuk ke Manajemen Kontak

Data Terupload & Ditambahkan ke Kontak

Step 2: Kirim Pesan
